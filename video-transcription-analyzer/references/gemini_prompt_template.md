# Шаблон промпта для Google Gemini

Использовать этот шаблон для создания промптов визуального анализа YouTube видео.

## Структура промпта

```markdown
# Промпт для Google Gemini: [Имя спикера] — "[Тема]"

**Тема выступления:** [тема]
**Спикер:** [имя] ([бэкграунд])
**День/время:** [если известно]

---

## Промпт для визуального анализа

\`\`\`
Это видео с [конференции/вебинара/интервью]. Спикер — [имя], рассказывает про [тема].

Мне нужно извлечь ВСЕ промпты и подходы, которые он показывает на экране.

Особенно обрати внимание на моменты, когда он демонстрирует:

1. **[Первый юзкейс]** — [что искать]
2. **[Второй юзкейс]** — [что искать]
3. **[Третий юзкейс]** — [что искать]
...

Для каждого момента, где видно текст промпта или код:
- Сделай скриншот/timestamp
- Выпиши промпт дословно
- Опиши контекст использования
- Отметь какие настройки видны

Также интересует:
- [дополнительные вещи для поиска]

Формат вывода:
---
**Timestamp:** [время в видео]
**Что показывает:** [описание]
**Промпт/код:**
\`\`\`
[точный текст]
\`\`\`
**Контекст:** [для какой задачи]
---
\`\`\`

---

## Что искать в видео (по транскрипту)

[Спикер] показывал:
- [пункт 1]
- [пункт 2]
- [пункт 3]

[Дополнительный контекст из транскрипта]

---

## Ключевые цитаты из транскрипта

> "[цитата 1]"

> "[цитата 2]"

---

*Файл для ресёрча в Google Gemini с YouTube видео*
```

## Принципы

1. **Конкретность** — Указывать что именно искать
2. **Контекст из транскрипта** — Помогает Gemini понять где искать
3. **Формат вывода** — Структурированный для удобства
4. **Цитаты** — Помогают найти нужные моменты
