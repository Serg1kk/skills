# Negative Patterns - words and patterns to exclude

This file is read by the Writer (Phase 4) and Rewriter (Phase 5). Contents are PROHIBITED in generated content.

---

## Prohibited Patterns

| Pattern | Example | Why |
|---------|---------|-----|
| "Before [bad]. Now [good]." | "Before I chose between X and Y. Now I load everything." | AI-typical contrast template. Sounds like an ad. If contrast is needed - show through a specific story, not a formula. |
| "It's not X - it's Y" (more than once) | "It's not more memory - it's a different mode" | Once is acceptable. When repeated - obvious generation marker. |

---

## How to use

1. Writer (Phase 4): check text for patterns from this file BEFORE sending to critics
2. Critic A (Phase 5): add this file to the checklist
3. Rewriter (Phase 5): remove found patterns

## Customization

Add your own prohibited words and patterns here. When generating content, ALWAYS read the current version of this file. Each project may have domain-specific words that sound unnatural (jargon, bureaucratese, etc.).
