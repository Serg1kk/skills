# LINKEDIN STYLE GUIDE TEMPLATE
## Guide for Writing LinkedIn Posts

**Customize this template for your profile.** Replace examples with your own.

---

## MAIN PRINCIPLE

**Write as an expert-practitioner sharing professional experience with colleagues.**

Not a corporate press release. Not a dry report. A specialist's living voice with real cases.

---

## POST STRUCTURE

### Typical structure:

```
[HOOK - first 2 lines visible before "...more", must grab attention]

[Context with professional accent - 2-3 sentences]

[What I did / observation - bullet points with arrows ->]

-

[Example from practice - specific case]

-> [Step/result 1]
-> [Step/result 2]
-> [Step/result 3]

-

[Conclusion / insight for professionals]

-

[Practical takeaway]
```

### Structure rules:
1. **HOOK** - first 2 lines are critical (visible before "...more")
2. **Separators** - use blank lines between semantic blocks
3. **Arrows** -> instead of bullets (read better in feed)
4. **Blank lines** - between every paragraph
5. **Ending** - practical takeaway (no cliche engagement questions like "Have you tried?")

---

## TONE OF VOICE

### LinkedIn-specific:

| Quality | How it shows | Example |
|---------|-------------|---------|
| **Professional** | Business/industry context | "As a PM with 16 years of experience..." |
| **Expert** | Specific numbers and tools | "Automated 3 processes in a week" |
| **Accessible** | Explains complex simply | No jargon, clear to wide audience |
| **Practical** | Takeaways for reader | "Conclusion for PMs/entrepreneurs" |

### Differences from Telegram:

| Telegram | LinkedIn |
|----------|----------|
| Maximum informal | Professional but not dry |
| Emoji: 1-3 | Emoji: 0-1 (or none) |
| Slang OK | No slang |
| Bullets | Arrows -> |
| No greetings | Can use "colleagues" |

## FORMATTING

### What works in LinkedIn:

```
Arrows instead of bullets:
-> First point
-> Second point
-> Third point

Separators between blocks:
-

Blank lines between paragraphs (mandatory!)
```

### Length:
- **Optimal:** 1500-2500 characters
- **Minimum:** 800 characters
- **Maximum:** 3000 characters

### Emoji:
- 0-1 per post (in title or accent)
- Can be fully without emoji
- Never spam

### Dashes:
- **NEVER use em-dashes** - it's an AI generation marker
- Use only short dash (-) or restructure the sentence
- Separators between blocks: use blank line instead

---

## HOOK FORMULAS

### Working formulas for first 2 lines:

```
[Observation/thesis]

[Contrast or amplification]
```

**Examples:**

```
There's a certain type of person in AI transformation.

They went through the whole path naturally.
```

```
Creating an MVP used to take weeks.

Now - a weekend. Here's how.
```

### DO NOT use:
- "Today I want to share..."
- "Friends! Important news!"
- "You won't believe, but..."
- Clickbait without substance

## POST TYPES

### 1. INSIGHT / Observation

```
[Thesis - 1-2 lines]

[Why this matters]

-

[Example from practice]

-> [Details]
-> [Details]
-> [Details]

-

[Conclusion]

-

[Practical takeaway]
```

### 2. CASE STUDY

```
[Result - what was achieved]

[Context - what was the task]

-

What I did:

-> [Step 1]
-> [Step 2]
-> [Step 3]

-

Result: [measurable outcome]

-

Takeaway for [your audience]:

[Practical conclusion]
```

## HASHTAGS

**Count:** 3-5 hashtags at the end of the post

**Selection algorithm:**
1. Extract key technologies/tools from the text
2. Add 1-2 broader tags for reach
3. Prefer specific tags over generic

**Format:**
```
-

#Tag1 #Tag2 #Tag3
```

---

## IMAGES

- Desirable for professional posts
- Increase reach

**What works:**
- Interface screenshots
- Before/After comparisons
- Process diagrams
- Clean illustrations (no memes)
